O backend foi construído usando Python 3.8
As dependências podem ser baixadas usando o comando
pip install -r .\requirements.txt
na pasta raiz de qualquer uma das duas aplicações.

Para executar cada aplicação é somente o comando 
python .\main.py
na pasta raiz

Existe uma biblioteca q permite o teste nas urls geradas adicionando '/docs'
